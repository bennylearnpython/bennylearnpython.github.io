使用数据处理函数

函数

和大部分计算机语言一样 sql也支持函数来处理数据 函数的在每个DBMS里面的不太相同，因此可移植性并不是很好。 所以为了代码可移植性，程序员不赞成使用函数。

使用函数

大多数sql实现支持以下类型的函数：

  1. 用于处理文本串的文本函数
  2. 用于数值数据上的数值函数
  3. 用于处理日期和时间的日期时间函数
  4. 返回DBMS正使用的特殊信息的系统函数

常用的文本处理函数

函数 说明
Left() 返回串左边的字符
Length() 返回串长度
Locate() 找出串的一个字串
Lower() 将串转换成小写
Trim() 去掉左右两边的空格
LTrim() 去掉左边的空格
RTrim() 去掉右边的空格
Right() 返回串右边的字符
Soundex() 返回串的soundex值
SubString() 返回字串的字符
upper() 串转换成大写

soundex()需要进一步解释。soundex是将文本串转换成为描述其语言表示的字母数字的模式的算法。 就是说soundex比较的串的发音而不是字母比较。

数值处理函数

数值处理函数仅仅处理数值数据，主要用于代数，三角和几何运算，因此没有串或者时间-日期处理函数这么频繁使用 但是在DBMS中反而是最统一的。

常用数值处理函数 | 函数 | 说明 | | ——– | ———– | | abs() | 一个数的绝对值 |
| cos() | 角度的余弦 |
| exp() | 一个数的指数 |
| mod() | 除操作的余数 |
| pi() | 返回圆周率 |
| rand() | 一个随机数 |
| sin() | 一个角度的正弦 |
| sqrt() | 一个数的平方根 |
| tan() | 一个角度的正切 |
