第二章：MYSQL简介

什么是mysql

mysql是一种DBMS， 是一种数据库软件 为什么大家都用mysql呢，基于以下几点：

  1. mysql是开源免费的
  2. 性能很好
  3. 可以信赖，很多大公司在用
  4. 易安装和使用

C/S软件

DBMS分为两类：

  1. 基于共享文件系统的DBMS（如microsoft access 和 filemaker）—桌面用途
  2. C/S数据库， 如mysql oracle —更高端

client 可以是MYSQL的工具，脚本，各种程序语言。 server 是MYSQL DBMS

MYSQL版本

介绍一下DBMS的版本 书中的内容有点老了，目前最新版本是mysql 8.0了

MYSQL工具

client工具，推荐了3种：

  1. mysql命令行

本地输入mysql，就会出现一个命令行，命令在mysql>之后，以；或者/g 结尾，help或\h来获得帮助，quit或者exit来退出命令行

可能要输入用户名，主机名，端口，密码等信息，用mysql –help来获取更多的参数

  1. MYSQL administrator

图形化交互

mysql administrator下载

  1. MYSQL Query Browser

也是一个图形化交互程序

mysql query browser下载
