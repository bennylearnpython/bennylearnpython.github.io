过滤数据

使用where子句

数据库一般包含大量的数据，很少需要检索所有行 所以通过where子句来对数据进行条件过滤 感觉跟grep的作用相似。

mysql> select name,help_topic_id from help_topic where help_topic_id = 100;
+----------+---------------+
| name     | help_topic_id |
+----------+---------------+
| MAKE_SET |           100 |
+----------+---------------+
1 row in set (0.00 sec)

mysql>

注意这里过滤的数据类型是数值型，所以可以不用’’

如果where和order by一起用的时候，要保证where在前，order by在后

where子句操作符

操作符 说明
= 等于
<> 不等于
!= 不等于
< 小于
<= 小于等于
> 大于
>= 大于等于
BETWEEN 在两个值之间
